# Fleet Apps catalog

Published **Fleet App Extension Protocol (FAEP)** packages that a running Fleet instance can install from `fleet.forgesdlc.com`.

## For operators

1. Open **`/admin/`** on your Fleet host and use the **Apps** tab to browse the catalog and install packages.
2. Each installed app appears as its own admin tab with a declarative UI and an in-package **docs wiki** at `/admin/apps/{id}/docs/`.
3. Read the per-app handbook page linked from the catalog for trust boundaries and prerequisites.
